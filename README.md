## Social Listening & Intent Analysis

*Tools that monitor the web (Reddit, Twitter/X, LinkedIn) to find people actively looking for your solution.*

  - [Agent-Reach](https://github.com/Panniantong/Agent-Reach) - MCP-compatible CLI that lets AI agents read and search Twitter, Reddit, YouTube, and GitHub without paid API keys.
  - [Awario](https://awario.com) - Brand monitoring and social listening tool that tracks mentions across the web and social networks.
  - [Buska](https://buska.io) - Mentions monitoring with sentiment analysis across social media and community platforms.
  - [F5Bot](https://f5bot.com) - Free keyword alert service for Reddit, Hacker News, and Lemmy.
  - [GummySearch](https://gummysearch.com) - Audience research tool for Reddit that surfaces communities and high-intent conversations around a topic.
  - [Leado](https://leado.co) - AI-powered Reddit lead generation tool that monitors subreddits for high-intent leads and writes personalized replies automatically.
  - [Scweet](https://github.com/Altimis/Scweet) - Python library for scraping tweets, profiles, followers, and following lists from Twitter/X without an API key.
  - [Syften](https://syften.com) - Keyword monitoring across Slack communities, Discourse forums, Reddit, and other platforms in real time.
  - [twikit](https://github.com/d60/twikit) - Async Python client for Twitter's internal API enabling tweet search, profile scraping, and timeline access without paid credentials.
  - [twscrape](https://github.com/vladkens/twscrape) - X/Twitter scraper with multi-account pooling that covers search results, user profiles, followers, and tweet metadata.
  - [URS](https://github.com/JosephLai241/URS) - Universal Reddit Scraper for archiving subreddits, comment threads, and redditor data via the PRAW API.

## Data Scraping & Enrichment

*Tools to turn a company name or social profile into a verified contact and enriched data set.*

### Email Discovery & Verification

  - [check-if-email-exists](https://github.com/reacherhq/check-if-email-exists) - Rust library with an HTTP backend that verifies whether an email address exists by checking DNS and SMTP without sending a message.
  - [email-sleuth](https://github.com/buyukakyuz/email-sleuth) - CLI tool that discovers and verifies professional email addresses from a person's name and company domain.
  - [MailChecker](https://github.com/FGRibreau/mailchecker) - Cross-language library for detecting disposable and throwaway email addresses, covering 55,000+ fake providers.

### LinkedIn & People Data

  - [LinkedInDumper](https://github.com/l4rm4nd/LinkedInDumper) - Python script to extract a structured employee list from any LinkedIn company page.
  - [linkedin-mcp-server](https://github.com/stickerdaniel/linkedin-mcp-server) - Open-source MCP server that gives Claude and other AI assistants access to LinkedIn profiles, companies, jobs, and direct messages.
  - [OpenOutreach](https://github.com/eracle/OpenOutreach) - AI-powered LinkedIn automation tool that finds leads based on a user-defined product description and target market.
  - [StaffSpy](https://github.com/cullenwatson/StaffSpy) - Python library to retrieve LinkedIn employee records including work history, education, skills, and contact information.

### Web, Maps & Enrichment Platforms

  - [Ai\_Web\_Scraper](https://github.com/akhilathuluri/Ai_Web_Scraper) - Streamlit, Selenium, and LangChain integration to extract specific website information bypassing CAPTCHAs.
  - [Apify](https://apify.com) - Cloud platform with a marketplace of pre-built scrapers for Google Maps, Instagram, TikTok, and dozens of other sources.
  - [Apollo](https://apollo.io) - B2B contact database of 275M+ records with built-in email finding, sequencing, and CRM enrichment features.
  - [Bright Data](https://brightdata.com) - Web data platform for large-scale structured data extraction with a global proxy network and dataset delivery API.
  - [Browse AI](https://browse.ai) - No-code tool to extract and monitor data from any website by training a robot on the target page.
  - [business-leads-ai-automation](https://github.com/asiifdev/business-leads-ai-automation) - Open-source Google Maps scraper paired with AI-generated email and WhatsApp outreach templates for SME prospecting.
  - [Clay](https://clay.com) - Data enrichment spreadsheet that pulls from 50+ providers and uses AI to research, score, and build lead lists automatically.
  - [Firecrawl](https://github.com/firecrawl/firecrawl) - API to search, scrape, and interact with the web for AI using natural language and markdown conversion.
  - [google-maps-scraper](https://github.com/omkarcloud/google-maps-scraper) - Python tool to extract 50+ data points from Google Maps listings including business emails, phone numbers, and social profiles.
  - [kaymen99/ai-web-scraper](https://github.com/kaymen99/ai-web-scraper) - LLM-powered web scraper built on Crawl4AI that extracts structured lead data from arbitrary website URLs.
  - [PhantomBuster](https://phantombuster.com) - Cloud automation tool for scraping data and performing sequences of actions on LinkedIn, Instagram, and Google Maps.
  - [ScraperAI](https://github.com/scraperai/scraperai) - AI-powered tool that simplifies web scraping by detecting page types, data fields, and generating reusable configs.
  - [WaterCrawl](https://github.com/watercrawl/WaterCrawl) - Self-hostable crawler that converts web pages into structured, LLM-ready Markdown or JSON output.
  - [WebScraping.AI](https://www.google.com/search?q=https://github.com/webscraping-ai/webscraping-ai-python) - Python client for API providing Chrome JS rendering and HTML parsing.

## Cold Outreach & Email AI

*Infrastructure and AI tooling for sending personalized emails at scale with high deliverability.*

  - [Instantly](https://instantly.ai) - Cold email platform with unlimited sending accounts and AI-powered warm-up to maintain inbox placement.
  - [Lavender](https://lavender.ai) - AI email coach that scores outbound messages and suggests real-time improvements inside Gmail and Outlook.
  - [Lemlist](https://lemlist.com) - Outreach platform that supports personalized images, videos, and multi-channel sequences in cold email campaigns.
  - [Postal](https://www.google.com/search?q=https://github.com/postalserver/postal) - Fully featured open source mail delivery platform for incoming and outgoing email.
  - [sales-outreach-automation-langgraph](https://github.com/kaymen99/sales-outreach-automation-langgraph) - LangGraph-based AI agent pipeline that automates lead research, qualification, and personalized messaging with HubSpot, Airtable, and Google Sheets integrations.
  - [Smartlead](https://smartlead.ai) - Cold email infrastructure focused on deliverability, multi-inbox rotation, and API-first workflow automation.

## Marketing Automation & CRM

*Tools and platforms for building automation workflows, managing campaigns, and tracking leads.*

  - [Chatwoot](https://www.google.com/search?q=https://github.com/chatwoot/chatwoot) - Open-source customer engagement suite to manage multi-channel conversations and capture leads.
  - [Mautic](https://www.google.com/search?q=https://github.com/mautic/mautic) - Open-source marketing automation project to manage campaigns, lead generation, and email tracking.
  - [n8n](https://www.google.com/search?q=https://github.com/n8n-io/n8n) - Workflow automation platform connecting APIs and services to build marketing and lead generation pipelines.
  - [Twenty](https://www.google.com/search?q=https://github.com/twentyhq/twenty) - Open-source CRM designed to track leads, customer interactions, and sales pipelines efficiently.

## Voice Agents & Calling

*AI that can initiate or handle phone conversations with prospects.*

  - [Bland AI](https://bland.ai) - Platform for building and deploying AI phone agents capable of handling complex, multi-turn inbound and outbound calls.
  - [Fireflies.ai](https://fireflies.ai) - AI meeting assistant that transcribes calls, analyzes conversation sentiment, and extracts CRM-ready action items.
  - [Synthflow](https://synthflow.ai) - No-code builder for deploying AI voice assistants for both inbound qualification and outbound prospecting.
  - [Vapi](https://vapi.ai) - Developer-focused voice AI infrastructure for building, testing, and scaling real-time conversational voice assistants.

## AI Copywriting & Personalization

*Tools that generate unique, research-backed messaging tailored to individual leads.*

  - [Jasper](https://jasper.ai) - Enterprise AI writer for producing marketing content with brand voice controls, templates, and team collaboration.
  - [Taplio](https://taplio.com) - AI content generation tool for building a personal brand and generating inbound leads on LinkedIn.
